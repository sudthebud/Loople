In Loople, a set of random words are connected in a loop, where the last letter of one word is the first letter of another. You only have a certain number of turns to guess all the words in the loop! Can you solve the loop, or will you be thrown for one?

Loople was originally created for the GMTK Game Jam 2025, where it achieved a rank of #275 in Creativity out of nearly 10,000 participants. This is a web based recreation of that game, intended to be a learning opportunity for web development.
